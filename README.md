# Quiz App Repository

This repository contains a simple quiz app codebase. 

## Getting Started

1. **Fork this repository**: Click on the "Fork" button in the top-right corner of this page to create your own copy of the repository.

2. **Modify questions**: Open `script.js` and change the questions according to your preferences. You can add, remove, or modify questions as needed.

3. **Update title**: Open `index.html` and change the title to represent your quizzes. This will appear in the browser tab when users access your quiz.

4. **Customize styles**: Optionally, you can customize the style of your quiz by editing `styles.css` in your forked repository.
